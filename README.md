# Athanasius Hacking Framework

## Installation
**1.Clone Repo**
```sh
git clone https://github.com/shyam999/Athanasius.git
```
**2.Install Requirements**
```sh
pip install -r requirements.txt
```
**3.Install hydra**
```sh
sudo apt install hydra
```
**4.Run The Script**
```sh
python athanasius.py
```
